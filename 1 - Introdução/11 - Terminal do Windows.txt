Terminal do Windows

Caso não tenha o terminal do windows instalado, instale-o através do link acima.

Ou

Utilizar o terminal Cmder (https://cmder.app/)
Este é um terminal que pode ser utilizado no Windows, Linux e Mac.
Ele simula o terminal do Linux no Windows, e é muito bom para quem está acostumado com o terminal do Linux.
